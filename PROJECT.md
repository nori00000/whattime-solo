# Project Metadata / 프로젝트 메타데이터

## Overview / 개요

WhatTime Solo is a personal scheduling service for a single host. It connects to
Google Calendar availability, exposes public booking links, creates calendar
events, and supports host/invitee cancellation flows.

WhatTime Solo는 1인 운영자를 위한 개인 일정 예약 서비스입니다. Google Calendar
가용 시간을 읽고, 공개 예약 링크를 제공하며, 캘린더 이벤트 생성과 호스트/초대자
취소 흐름을 지원합니다.

## Purpose / 목적

- Reduce back-and-forth scheduling for solo operators.
- Make public booking flows small enough to run as an independent product.
- Keep planning, API, operations, and deployment notes close to the code.

- 1인 운영자의 반복적인 일정 조율 부담을 줄입니다.
- 독립 제품으로 운영 가능한 작은 공개 예약 흐름을 제공합니다.
- 제품 계획, API, 운영, 배포 문서를 코드와 함께 관리합니다.
